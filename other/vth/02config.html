                  配置 VTH

                  AckervenVuePress Theme Hope约 884 字
                  此页内容
                  • 配置参考

                  这节将介绍 VTH 的配置文件在哪里，如何对 VTH 进行自定义。

                  关于配置文件的介绍以及配置的作用域查看 VuePress Theme Hope → 小白教程 → 项目配置open in new window 。

                  下面流程基于 v2.0.0-beta.168，新的版本可能会有变化。

                  在配置 VTH 过程中，会修改以下文件：

                  • .vuepress/config.ts：站点配置
                  • .vuepress/theme.ts：主题配置
                  • .vuepress/navbar/zh.ts：导航栏配置（主题配置的一部分）
                  • .vuepress/sidebar/zh.ts：侧边栏配置（主题配置的一部分）

                  1. 移除多语言配置

                    移除多语言，需要做以下工作：

                    • 移除配置文件中关于 locales 相关的配置
                  • 把 src 目录下所有 Markdown 文件删掉（zh 目录除外）
                    • 把 src/zh 目录的内容复制到 src 目录中，再删掉 src/zh 目录
                    • 把 README.md 中的 /zh/ 替换成 /
                    • 把 navbar/zh.ts 和 sidebar/zh.ts 中的 /zh/ 替换成 /

                  1. 确定导航栏和侧边栏的内容

                    共设定八个分类：

                    • 基石（cornerstone）：数学、计算机基础学科…
                    • 语言（language）：编程语言
                    • 框架（framework）：主流开发框架
                    • 算法（algorithm）：数据结构、算法…
                    • 技术（technology）：工具、技术、操作系统…
                    • 哲学（philosophy）：软件工程、设计模式、编程思想…
                    • 随笔（article）：博客、奇淫技巧…
                    • 其他（other）：不便分类的文章

                  2. 准备好静态资源以及 Markdown 文件

                    需要准备的静态资源：

                    • favicon.ico
                    • logo.png
                    • logo.svg

                    同时需要为每个分类准备一个文件夹和 README.mdopen in new window 文件，以及主页的 README.mdopen in new window 和目录 summary.mdopen in new window 文件。

                  3. 按照 VuePress Theme Hope → 指南open in new window 配置主题

                  4. 按照 VuePress2 → 参考 → 配置open in new window 配置站点

                  5. 按照 VuePress Theme Hope → 配置open in new window 配置其他的内容（可选）

                  6. 其他配置

                    • 图片圆角显示

                      修改 index.scss 文件，添加以下内容：

                      img {
    border-radius: 5px;
}

                  • config.ts

                    详情
                    import { defineUserConfig } from "vuepress";
import { searchProPlugin } from "vuepress-plugin-search-pro"
import theme from "./theme.js";

export default defineUserConfig({
  base: "/",
  lang: "zh-CN",
  title: "Notebook",
  description: "Ackerven's knowledge planet.",

  plugins: [
    searchProPlugin({
      indexContent: true,
      customFields: [
        {
          getter: (page) => page.frontmatter.category,
          formatter: "分类: $content",
        },
        {
          getter: (page) => page.frontmatter.tag,
          formatter: "标签: $content",
        },
      ],
    }),
  ],

  theme,

  shouldPrefetch: false,
});
