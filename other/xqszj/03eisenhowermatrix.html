                                            # 什么是四象限法则

                                            四象限法则是由史蒂芬·科维 (Stephen R. Covey) 提出的时间管理理论，这个理论按照重要性和紧迫程度将事情划分到四个象限：重要而且紧急、重要但不紧急、不重要但紧急、不重要而且不紧急。

                                            重要性评判标准是价值观，紧迫程度评判标准是截止时间。

                                            提示

                                            上一篇文章 职业价值观 的附录有职业价值观自测量表，这个结果可能仅适用于处理工作上的事情，生活与工作也许是两个不同的维度，对于生活上重要的事情，需要根据自己的经验或其他的价值观进行评估。

                                            • 第一象限：重要且紧急

                                              这个象限里面的事情需要我们立即去处理。

                                              例子：快到截止时间的任务

                                              在这个象限，我们需要思考：真的有那么多重要而且紧急的事情吗？

                                            • 第二象限：重要但不紧急

                                              这个象限里面的事情需要及时处理，否则随着时间的推移，这个象限的事情就会进入到第一象限。

                                              例子：距离截止还有一段时间的任务

                                              在这个象限，我们需要思考：如何避免更多的事情进入令人讨厌的第一象限？

                                            • 第三象限：不重要但紧急

                                              需要明白的是，紧急不代表是重要的，处理这个象限里面的事情所花费的精力是相当无奈的。

                                              例子：不速之客的来访

                                              在这个象限，我们需要思考：如何尽量减少第三象限的事情？

                                            • 第四象限：不重要而且不紧急

                                              这个象限里面的事情，可以看作是前三个象限的调剂，我们在适当放松的时候可以进入这个象限，但不能一直呆在这个象限，否则就是混日子。

                                              例子：看无聊的电视节目，一个人闲逛

                                              在这个象限，我们需要思考：工作中是否有必要进入这个象限？

                                            # 将事情放入四个象限

                                            要将手头的事情归类到四个象限，可以根据以下步骤：

                                            1. 将需要完成的事情列出来
                                            2. 根据你的价值观，为每件事情标上“重要”或“不重要”
                                            3. 根据事情的截止日期，为每件事标上“紧急”或“不紧急”
                                            4. 最后为相同属性的事情（e.g. 重要和紧急的事情）标上优先级“高”、“中”和“低”

                                            # 处理四个象限的原则

                                            将事情分类到四个象限后，对每个象限的事情应该采取不同的态度和处理方法。

                                            # 第一象限

                                            立即去做。

                                            这个象限里面的事情 80% 都是源自我们在第二象限没有处理好的事情。它像一片雷区，也是我们主要压力和危机感的来源，这个象限进入的次数越少越好。

                                            # 第二象限

                                            有计划去做。

                                            这个象限是我们最应该投资时间的。我们应该第一时间对第二象限的任务进行分解，并制定时间表。这样，我们每次投入很少的时间，就完成了一项庞大的任务。

                                            # 第三象限

                                            交给别人去做。

                                            我们应该清楚地认识到第三象限的事情是我们忙碌而且盲目的源头。这个象限的事情最好能放权给别人去做，如果我们是基层的执行者，我们尽量委婉地拒绝。

                                            # 第四象限

                                            尽量不去做。

                                            这个象限里面的事情是用来缓冲调整的，当我们感到疲惫的时候，可以做一些这里面的事情放松身心，但不能在里面投入太多的精力，否则你就是在浪费生命，不务正业了。

                                            # 怎么走出第三象限

                                            我们现在知道走出第一象限的方法是集中精力处理好第二象限的事情，那么如何走出第三象限呢？

                                            我们可以使用“猴子法则”让自己摆脱第三象限。

                                            猴子法则是比尔·翁肯 (Bill Oncken) 提出的一个理论，叫“背上的猴子”。他有一次偶然发现，自己忙于加班的时候，下属在悠哉游哉的打着高尔夫球。这让他意识到，主管人员之所以时间不够用，一个很重要的原因是没有做好授权分责，将太多本该下属做的工作招揽到自己的身上了，以至于永远在苦苦地追赶工作进度。

                                            使用猴子法则的时候，有两点需要注意的：

                                            • 明确职责

                                              我们需要明确自己的责任边界，确定这只猴子不是自己的。否则，我们把本该自己负责的事情推掉就是在逃避责任了。

                                            • 注意沟通方式

                                              对于不是我们的猴子，我们要学会说“不”，我们要明确，坚决的拒绝别人把猴子甩给我们。我们既要注意沟通方式，委婉的拒绝，又要做到在拒绝别人的时候不要含糊不清。

                                            如果你的上头一定要把猴子甩给你，一定要沟通清楚他对这个事情的想法，还有预期结果，不要不明不白的接下来。

                                            一个有趣的例子

                                            试想一下这样的情况：你在走廊上碰到一位同事，他说：“我遇到麻烦了，能不能和你谈谈？” 于是你开始关切地听他讲述。结果这个问题果然很麻烦，你听了半个小时才弄清楚是怎么回事，而且还没法立刻给出建议，于是你说：“这个问题很复杂，我现在没有时间和你讨论，让我仔细想想，回头咱们再谈谈。”

                                            让我们用猴子理论来分析一下这个过程。首先，你和同事在走廊上偶遇之前，谁的背上有猴子？显然，是同事的背上。猴子不在你的背上，你甚至不知道有这只猴子存在。接着，你开始倾听同事的讲述，这时同事背上的猴子悄悄向你的背上跨过来一只脚。然后，听完同事的讲述，你表示要仔细想想再和同事讨论。这时，猴子便完全转移到了你背上。你接过了同事背上的猴子，而同事则变成了监督者。此后，他会不时跑来问你：“那件事你考虑得怎么样了？” “我们什么时候再谈谈？”……

                                            我们虽然应该帮助遇到困难的同事，但是也应该避免这样的情况：让他们把你当作他们自己猴子的收容站，你收的愈多，他们给的愈多，到最后你被堆积如山的别人的问题所困扰，甚至没有时间照顾自己的“猴子”。你将一些并非自己职责的事情做得很有效率，可这值得你沾沾自喜吗？你自己的事情完成得怎么样呢？这就是猴子法则，应用它的前提是你要明确自己的责任边界。

                                            # 保持在第二象限

                                            第二象限是四象限的核心和最终目的，我们平时要围绕第二象限展开我们的工作。那么，怎么做到将自己精力投资在第二象限呢？

                                            我们可以对第二象限的事情进行目标描述和任务分解。这样做的好处如下：

                                            • 迫使我们将精力花在第二象限，这在一开始是非常重要的。
                                            • 有利于将一个项目做细，做得有计划，时刻知道你下一步该做什么，也更加有行动力。
                                            • 明确任务完成的标准，当达到标准的时候就可以放下你的心头大石，减轻你的压力。
                                            • 分成小的任务后有利于进度控制，你可以明确地知道该任务是否造成了拖延，并立即调整后面任务的时间规划。

                                            对事情进行目标描述和任务分解可以有效消除时间管理的三大杀手——信息不够、拖延、预期结果不明确。
